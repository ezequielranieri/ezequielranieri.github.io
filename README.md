# Ezequiel Ranieri Portfolio

Professional portfolio website for Ezequiel Ranieri, a self-taught Python developer specializing in backend engineering and cybersecurity.

## 🚀 Live Demo
**View the live site:** [https://ezequielranieri.github.io](https://ezequielranieri.github.io)

## 💻 Tech Stack
Built for speed and simplicity with a **single-file architecture**:
- **Framework**: [React 18](https://reactjs.org/) (via CDN)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- **Compilation**: [Babel](https://babeljs.io/) (via CDN)
- **Deployment**: [GitHub Pages](https://pages.github.com/)

## 🛠 Features
- **Cyber-Terminal Aesthetic**: Custom CRT scanlines, noise effects, and grid backgrounds.
- **Dynamic Animations**: React-driven typing effects, glitch hover states, and scroll reveals.
- **Mobile Responsive**: Fully optimized for all screen sizes.
- **Zero Build Step**: The entire site is contained in a single `index.html` file, requiring no local installation or build process.
